# Frontend-test
Тестовое задание
Demo: https://dmazanov.github.io

### Установка
```
1. git clone https://github.com/dmazanov/frontend-test.git
2. cd frontend-test
3. bower i
4. npm i
5. для автоматической компиляции изменений в проекте:
	gulp
6. для сборки проекта в папку dist:
	gulp build
7. для очистки папки dist:
	gulp clean
```
